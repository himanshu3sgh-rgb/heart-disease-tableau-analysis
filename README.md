
# Heart Disease Analysis using Tableau

## Overview
This project analyzes heart disease data using Tableau dashboards and a Flask web interface.

The goal is to visualize key risk factors such as:
- Age
- Cholesterol
- Gender
- Lifestyle indicators

## Technologies
- Python
- Flask
- Pandas
- Tableau
- HTML/CSS

## Project Structure
heart-disease-tableau-project/
│
├── data/
├── templates/
├── static/
├── tableau_dashboard/
├── app.py
├── requirements.txt
└── README.md

## Run the Project

Install dependencies

pip install -r requirements.txt

Run

python app.py

Open in browser

http://127.0.0.1:5000
